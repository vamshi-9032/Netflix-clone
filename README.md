# 🎬 Netflix Clone

This is a Netflix India landing page clone built using only HTML and CSS.  
The main aim of this project is to practice frontend design and recreate a real-world website layout without using JavaScript.

---

## ✨ Features

- 🎨 Hero section with poster-style background and overlay
- 🌈 Curved separator created using CSS
- 🔥 Trending movies section with scrollable cards
- 📋 Feature cards explaining platform benefits
- ❓ FAQ section using pure CSS (checkbox method)
- 🦶 Footer with multiple links and language option
- 📱 Fully responsive design (desktop, tablet, mobile)
- 🎞️ Smooth hover effects and animations

---

## 🛠️ Tech Stack

- HTML5  
- CSS3 (Flexbox, Grid, Animations)  
- Google Fonts (Inter, Bebas Neue)  

🚫 No JavaScript used
